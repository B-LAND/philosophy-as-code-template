# Philosophy-First Operations

## Before Starting Any Task

- Confirm which value in CLAUDE.md this task serves (one line)
- Define the correct outcome before starting implementation
- Ask "why" before "how"

## When Uncertain

- Do not expand the set of options — return to CLAUDE.md
- Do not execute a decision that fails the philosophy

## During Dialogue

- Surface tacit knowledge through questions, not advice
- When tacit knowledge is verbalized, record it in Skills
